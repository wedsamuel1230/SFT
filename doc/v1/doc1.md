## **1. 產品形態與穿戴設計**

### **硬體外形概念**

SmartPing Coach 包含兩個核心元件：

**元件 A：球拍感應器（Racket Sensor Pod）**

generated_image:154

**產品規格**

- **尺寸**：35mm × 25mm × 12mm（信用卡厚度）
- **重量**：18-25g（不影響揮拍平衡）
- **材質**：3D 列印聚酯或鋁合金外殼
- **安裝位置**：球拍柄下方，靠近手腕處（利用重力和旋轉最高點）
- **固定方式**：
    - 選項 1：磁吸（側邊內藏磁鐵，可快速拆卸）
    - 選項 2：魔術貼（可調整位置，不損傷球拍）
    - 選項 3：專用夾具（可適配多種球拍）

**內部硬體**

```

├──────────────────────────────┤

│  ├─ 綠色：已連接              │

│  ├─ 3 軸陀螺儀               │
│  ├─ 3 軸加速度計              │
│  └─ I2C 介面                 │
│                              │
│  下層：ESP32 + 電池           │
│  ├─ 微控制器                 │
│  ├─ 藍牙 BLE 晶片            │
│  ├─ 3.7V 500mAh 電池        │
│  └─ 充電接口（USB-C）        │
└──────────────────────────────┘
```

**元件 B：手機應用（SmartPing App）**

- iOS 13+ 或 Android 8+
- 大小：~50MB（輕量級）
- 需藍牙 4.0+ 支援

## **3. 核心使用方式與互動模式**

### **模式 A：自由訓練模式（Auto Mode）**

```
用戶按「開始訓練」
     ↓
進入被動監聽狀態
     ↓
感應器持續監控陀螺儀信號
     ↓
每當陀螺儀峰值 > 50°/s 時觸發分類
     ↓
App 即時顯示動作類型 + 分數 + 語音反饋
     ↓
用戶無需任何額外操作，專注訓練
```

**使用場景**：自主訓練、練習某種技術動作

***

### **模式 B：課程模式（Course Mode）**

```
用戶選擇「課程」頁面
     ↓
選擇預設課程，例如：
  └─ 「正手攻練習」
     目標：30 分鐘內，完成 100 次正手攻
     成功條件：平均分 > 75 分
     
     ↓
開始課程計時
     ↓
每次擊球都記錄並比對課程目標
     ↓
進度條實時更新
  └─ 已完成：45/100
     已用時間：15:30 / 30:00
     當前平均分：78 分
     
     ↓
課程完成或時間到
     ↓
顯示課程成績報告 + 獎勵
  └─ 🎯 課程達成！
     完成次數：102
     平均分：79.2
     進度：🌟🌟🌟
```

**預設課程範例**


| 課程名稱 | 目標 | 時長 | 難度 |
| :-- | :-- | :-- | :-- |
| 正手基礎 | 50 次正手，分數 >70 | 10 min | ⭐ |
| 反手防守 | 80 次反手 push，穩定性 | 20 min | ⭐⭐ |
| 發球強化 | 30 個發球，速度 >200°/s | 15 min | ⭐⭐⭐ |
| 綜合訓練 | 隨機 100 次擊球 | 30 min | ⭐⭐⭐ |
| 對標教練 | 與標準動作對齊 | 自訂 | ⭐⭐⭐⭐ |


***

### **模式 C：對標模式（Benchmark Mode）**

```
用戶選擇「對標教練」
     ↓
App 播放「標準正手動作」影片（2 秒）
     ↓
用戶嘗試複製此動作
     ↓
感應器捕捉用戶動作的陀螺儀特徵
     ↓
與標準動作特徵進行 DTW 比對
     ↓
顯示相似度評分：
  ├─ 90-100%：完美複製 ✓✓✓
  ├─ 70-89%：非常接近 ✓✓
  ├─ 50-69%：有差距 ✓
  └─ <50%：需改進
     
     ↓
給出精確反饋：
  「你的旋轉強度略低（當前 80°/s vs 標準 120°/s）
   建議增加腕部發力」
```

## **9. 快速參考：SmartPing Coach 基本操作**

### **新用戶快速上手指南**

| 步驟 | 操作 | 預期結果 |
| :-- | :-- | :-- |
| **第 1 步** | 充電感應器（USB-C，30 分鐘充滿） | LED 藍色 → 綠色 |
| **第 2 步** | 用魔術貼貼在球拍柄下方 | 感應器牢固，不晃動 |
| **第 3 步** | 打開手機藍牙 \& App | App 自動掃描附近設備 |
| **第 4 步** | 選擇「SmartPing-XXX」配對 | LED 變綠，App 顯示「已連接」 |
| **第 5 步** | 輸入名稱 \& 慣用手 | 保存個人資料 |
| **第 6 步** | 按「開始訓練」 | 進入訓練模式，可開始擊球 |
| **第 7 步** | 做 5-10 個擊球動作 | App 開始識別 \& 反饋 |
| **第 8 步** | 訓練完成按「結束」 | 自動生成成績報告 |

### **常見問題與解決**

| 問題 | 解決方案 |
| :-- | :-- |
| App 無法連接感應器 | ✓ 檢查藍牙是否開啟<br>✓ 重新配對<br>✓ 重啟感應器（關-開） |
| 分數總是很低 | ✓ 確認感應器位置是否貼牢<br>✓ 做大幅度擊球（小動作無法偵測）<br>✓ 檢查感應器電量 |
| 語音反饋延遲 | ✓ 靠近手機（BLE 有效距離 10-50m）<br>✓ 減少附近其他藍牙設備干擾 |
| 電量耗盡過快 | ✓ 關閉不必要功能<br>✓ 降低採樣率（設置 → 性能）<br>✓ 定期充電 |
