# SmartPing Coach｜基於陀螺儀感測器的乒乓球訓練系統

## 設計概念

### 設計靈感來源
- 目標硬體成本約170港元、完全離線、即時回饋。
- 普惠動機：全球約有3億乒乓球愛好者，但多數人無法負擔高端訓練設備。
- 教育價值：作為優秀的 STEM 教育案例，涵蓋物理（慣性測量）、電子工程（感測器電路）、電腦科學（機器學習、藍牙通訊）、數學（特徵提取、決策樹演算法）。
- 可遷移性：低成本 IMU + 邊緣運算的技術框架可啟發其他運動（網球、羽毛球、高爾夫等）的穿戴式訓練系統。

### 設計作品時的思考過程
- 架構比較：團隊評估了「純邊端」、「邊端 + 手機混合」、「雲—邊—端三層」三種方案。
- MVP 抉擇：最終選擇邊端 + 手機混合方案作為 MVP，機器學習運算在手機完成，邊端僅負責傳送數據。
- 指標設定：延遲 < 100 毫秒、精度目標，以實用性、可達性與隱私安全為優先（即使精度略低於視覺系統）。

### 遇到的挑戰與解決方案
- 穿戴舒適性與球拍平衡：
	- 安裝位置：球拍柄下方靠近手腕，既能捕捉最大旋轉角速度，又不影響揮拍平衡。
	- 固定方式：
		- 專用夾具式（通用性強但成本較高）
	- 外殼與重量：3D 列印外殼，尺寸約 35×25×12 mm，重量 18–25 g，確保輕量化不影響發揮。
- 感測器位置敏感性：安裝偏差會影響數據品質。解法：提供標準化安裝指引與位置校準程序，並建議採用魔術貼以便微調。
- 動作類型覆蓋不全：目前主要識別正手攻、反手推、發球等。解法：擴增訓練數據集、提升動作類別；評估導入 TensorFlow Lite 輕量 CNN 以強化複雜技術（如削球、吊球）的辨識。
- 電池續航：500 mAh 鋰聚合物電池約 8–10 小時；可升級至 1000 mAh 以延長至 15–20 小時。

## 作品介紹

### 作品特點
- 核心創新：將邊緣運算機器學習與低成本 IMU 感測器結合，打造「邊端決策樹 + 低功耗藍牙（BLE）」的乒乓球動作分類系統。
- 系統構成：
	- 球拍智能感應器（MPU‑6050 + ESP32 等低功耗硬體）
	- 手機應用程式（負責視覺化與紀錄，演算法於端上執行）
- 即時功能：
	- 動作即時分類（識別發球、正手攻、反手推、推擋等）
	- 質量評分（1–10 分制，基於陀螺儀峰值、加速度、一致性計算）
	- 訓練日誌（擊球數、平均分、動作分布）
	- 進度曲線（過去 7 天表現變化）
- 可負擔性與可製作性：單套硬體成本約 170 港元；元件通路成熟，社群資源豐富，中學生可透過 Arduino IDE 輕鬆開發與製作。

### 可行性分析
- 硬體成熟度高：MPU‑6050 與 ESP32 為市場成熟產品，供應鏈穩定、開發生態活躍。
- 可靠度與取樣率：文獻與實務均證實 MPU‑6050 在運動追蹤中的可靠性；取樣率可達 1000 Hz，足以捕捉乒乓球快速動作。
- 邊緣運算效能：端上決策樹推論可在 < 100 ms 內完成，滿足即時回饋需求；目標精度 70–80%。
- 開發門檻低：Arduino IDE 與豐富開源資源有助於快速迭代與教育推廣。

### 預期影響
- 對使用者與教練：提供數據驅動的訓練管理；可匯出 CSV 供教練分析進步、識別弱點、制定個人化訓練計畫；量化數據提升客觀評估與資源配置效率。
- 可負擔性與普及：170 港元大幅降低門檻，適用於學校、社區中心、家庭，尤其資源匱乏地區。
- 應用場景擴展：
	- 其他球拍運動（網球、羽毛球、壁球等）：調整安裝位置與模型即可遷移。
	- 康復醫療：用於上肢康復動作追蹤與量化進度評估。
- 商業化潛力：若定價 500–800 港元（仍低於視覺系統年訂閱費 ~1000 港元），以香港校隊、業餘球隊、球館為目標客群；保守估計年銷售 500 套可創造約 30 萬港元營收。未來可拓展雲端教練平台、多球員對戰模式、AI 個人化訓練計劃等增值服務。
- 開源與人機協作：在 GitHub 開源硬體設計、韌體、模型與 App，促進全球社群共創；AI 提供量化分析，輔助而非取代教練，形成「人類專業判斷 + AI 數據分析」的互補關係。
- 趨勢展望：隨著 TinyML 技術進步、感測器成本下降與開源生態完善，穿戴式訓練設備將更普及、智能、個人化。

能否利用低成本的慣性測量單元(IMU)感測器,結合邊緣運算機器學習技術,創造一個成本約170港元、完全離線、實時反饋且中學生可自製的乒乓球訓練系統。

技術路徑選擇階段,團隊比較了純邊端方案、邊端+手機混合方案、雲邊端三層方案三種架構。最終選擇純邊端方案作為MVP(最小可行產品),
所有機器學習運算在微控制器本地完成,延遲<100毫秒,精度目標70-80%,手機僅負責顯示與數據記錄。此方案雖精度略低於視覺系統,但在實用性、可達性、隱私安全方面全面勝出。

挑戰三：穿戴舒適性與球拍平衡
感測器安裝位置直接影響數據準確性與用戶體驗。安裝於球拍柄下方靠近手腕處,既能捕捉最大旋轉角速度,又不影響揮拍平衡。設計團隊提出三種固定方式：磁吸式(快速拆卸)、魔術貼式(可調位置且不損傷球拍)、專用夾具式(通用性強但成本較高)。感應器外殼尺寸35mm×25mm×12mm,重量18-25克,通過3D列印製作,確保輕量化不影響球員發揮。

SmartPing Coach 的核心創新在於將邊緣運算機器學習與低成本IMU感測器結合,創造首個「邊端決策樹+低功耗藍牙」的乒乓球動作分類系統。系統包含兩大核心元件：球拍智能感應器與手機應用程式

創新功能包括：動作即時分類(識別發球、正手攻、反手推、推擋等)準確度,質量評分(1-10分制,基於陀螺儀峰值、加速度、一致性計算),訓練日誌(記錄擊球數、平均分、動作分布),進度曲線(過去7天表現變化)。
